# Курсовой проект 2 (Поиск вакансий)

Учебный проект, посвящён модулю - 
"Объектно-ориентированное программирование"

### Функционал

* Поиск вакансий, на ресурсе hh.ru
* Фильтрация по ключевым словам
* Сравнение вакансий между собой
* Работа с файлами


## Тестирование

### Покрытие кода тестами

Для проверки покрытия кода тестами можно использовать `pytest-cov`:

```sh
pytest --cov=src
```

### Логирование и отчеты

Для генерации HTML-отчёта о покрытии выполните:

```sh
pytest --cov=src --cov-report=html
```
